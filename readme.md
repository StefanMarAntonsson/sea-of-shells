# Sea of Shells

Collection of niche scripts.

## What’s in here?
Little utilities and one-off scripts I build for my own setup (and sometimes others).

| Directory | What it is |
|-----------|------------|
| [apps-fzf](apps-fzf/) | Browse installed pacman packages and Flatpaks in fzf with a preview; open full details in `less`. |
| [spotify-status](spotify-status/) | Waybar module script: current track, artist, time, and pause state via `playerctl`. |
| [start-in-scratchpad](start-in-scratchpad/) | Hyprland helper to launch or refocus an app in the special scratchpad with a fixed size and position. |
| [yoink-steam](yoink-steam/) | Hyprland: move the Steam window to the active workspace and open the library. |

Each folder has its own readme with requirements and setup.
